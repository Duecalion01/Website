# Football Site

A simple static website for sharing football (soccer) news, scores, and highlights.

## Features
- Static HTML/CSS/JavaScript site
- Lightweight and easy to host (GitHub Pages, Netlify, or any static host)

## Getting Started

No build tools required — open the site locally or serve with a minimal static server.

Quick local preview using Python 3:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## File Structure
- [index.html](index.html) — main page
- [styles.css](styles.css) — styles
- [script.js](script.js) — JavaScript behavior
- [SETUP.md](SETUP.md) — additional setup notes

## Usage
- Edit `index.html` to change page content.
- Update `styles.css` to tweak appearance.
- Modify `script.js` to change interactivity or add features.
